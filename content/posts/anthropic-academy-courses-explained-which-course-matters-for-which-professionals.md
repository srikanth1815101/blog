# Introduction

Artificial intelligence is rapidly becoming a **core layer of modern software systems**. Organizations are integrating AI into:

* enterprise applications
* developer workflows
* cloud platforms
* analytics systems
* automation pipelines

To help professionals learn how to use these technologies responsibly and effectively, **Anthropic launched Anthropic Academy**, a structured learning platform focused on real-world AI skills and the Claude ecosystem.

The academy includes courses covering:

* AI collaboration and fluency
* developer tooling
* AI application development
* enterprise AI integration
* cloud AI platforms

However, each course targets **different types of professionals**. Some are designed for **software developers**, while others are meant for **educators, students, or organizations adopting AI tools**.

Understanding which course is relevant for your role can help you **prioritize learning and build practical AI capabilities faster**.

- - -

# Claude Code in Action

## What the Course Teaches

This course focuses on integrating **Claude Code** into a real development workflow.

Developers learn how AI can assist with:

* writing code
* debugging issues
* refactoring large codebases
* generating documentation
* navigating unfamiliar projects

It demonstrates how Claude interacts with repositories, context files, and development environments.

## Who Should Take This Course

This course is highly relevant for:

* Backend developers
* Frontend developers
* Full-stack developers
* DevOps engineers
* Engineering teams adopting AI coding assistants

## Why It Matters in Enterprise Development

In large organizations, developers often work with **complex and legacy codebases**.

AI coding tools help developers:

* understand existing systems faster
* automate repetitive coding tasks
* reduce debugging time
* improve developer productivity

For teams adopting **AI-assisted software development**, this course is extremely valuable.

# Claude 101

## What the Course Teaches

Claude 101 introduces the **core capabilities of Claude models** and how they can be used in everyday professional tasks.

Topics include:

* interacting with Claude
* writing effective prompts
* understanding AI capabilities and limitations
* practical productivity workflows

The course focuses on **AI usability rather than deep technical implementation**.

## Who Should Take This Course

Claude 101 is ideal for:

* professionals new to AI tools
* developers exploring AI platforms
* product managers
* analysts and consultants
* knowledge workers

## Why It Matters in Enterprise Environments

Many organizations face challenges with **unstructured AI usage**.

Employees use AI tools but often lack guidance on how to use them effectively.

This course helps professionals:

* learn structured prompting
* understand AI limitations
* avoid misuse of AI tools
* improve everyday productivity

For developers, it serves as a **foundation before moving to advanced AI engineering courses**.

# Introduction to Model Context Protocol (MCP)

## What the Course Teaches

Model Context Protocol (MCP) enables AI models to connect with **external tools, services, and data sources**.

This course teaches developers how to:

* build MCP servers
* create MCP clients
* connect AI models with external APIs
* integrate AI with databases and services

The course focuses on three MCP primitives:

* tools
* resources
* prompts

## Who Should Take This Course

Most relevant for:

* backend developers
* AI engineers
* platform engineers
* developers building AI assistants

## Enterprise Benefits

Enterprise AI systems must interact with **real organizational data**.

Examples include:

* CRM systems
* document repositories
* analytics platforms
* internal APIs

MCP allows AI models to **securely access and interact with these systems**, making it essential for enterprise AI development.
